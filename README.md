# 🏠 Airbnb Data Analysis Project  

## 📌 Project Overview  
This project analyzes Airbnb dataset to uncover insights about **customer behavior, pricing trends, and neighborhood performance**.  

The main objective is to solve a real-world business problem:  
👉 *Why are Airbnb sales declining?*  

---

## 🎯 Problem Statement  
Airbnb listings are experiencing a decline in bookings.  
This project aims to identify the root causes using data analysis.

---

## 📊 Dataset Features  
- 🏙 Neighbourhood → Location of listing  
- 💰 Price → Cost per booking  
- 👥 Accommodates → Capacity of property  
- 📅 Date → Booking timeline

---

## 📈 Key Insights  

✔ High-demand neighborhoods drive most bookings  
✔ Customers prefer specific room types (especially luxury)  
✔ Pricing directly impacts booking rates  
✔ Certain days show peak booking activity  
✔ Sales decline is caused by:  
- Overpricing in low-demand areas  
- Low visibility in certain neighborhoods  

---

## 💡 Business Impact  

This analysis helps to:  
- Optimize pricing strategy  
- Improve customer targeting  
- Increase occupancy rates  
- Identify high-performing locations
